# Learnova | AI-Powered "Zero to Hero" Learning Platform

Learnova is an intelligent educational ecosystem that leverages the **Gemini API** to transform any topic into a complete, structured learning journey. From absolute beginner to advanced mastery, Learnova automates the curriculum so learners can focus on growth.

## 🚀 Live Application
**Experience the future of learning:** [learnova-chi.vercel.app](https://learnova-chi.vercel.app/)

---

## 🧠 AI-Driven Features (Powered by Gemini)

### 🗺️ "Zero to Hero" Roadmaps
Generates structured learning paths from scratch. Tell Learnova what you want to learn, and it builds the entire curriculum, providing curated materials to take you from a novice to a pro.

### 🕸️ Dynamic Mind Maps
Visualizes complex subjects through AI-generated mind maps, allowing students to see the "big picture" and how different concepts interconnect.

### 📝 Smart Notes & Custom Length Answers
Get explanations on your terms. Whether you need a 50-word summary or a deep-dive 1000-word analysis, Learnova tailors its responses to your specific wording limits.

### 📝 Interactive Quizzes
Automatically generates targeted assessments based on your learning materials to test retention and identify knowledge gaps.

---

## ✨ Creative & Analytical Features

### ⚛️ Physics-Based Animation Engine
Built with a "touch of creativity," the platform uses physics-driven interactions for a tactile, immersive web-surfing experience that keeps learners engaged.

### 📊 Performance Analytics
Includes real-time performance graphs and DSA-style statistics to track your learning velocity and consistency over time.

---

## 🛠️ Technical Architecture

* **AI Engine:** Google Gemini API
* **Framework:** React + TypeScript + Vite
* **Styling:** Tailwind CSS + shadcn/ui
* **Motion:** Physics-based interactive logic
* **Deployment:** Vercel
